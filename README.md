# AFL Championship Manager

A browser-based AFL manager game built with vanilla HTML/JS.

Play as the Geelong Cats through a full 22-round season, finals series, and rookie draft.

## Features
- Full 22-round fixture with live match feed
- Team selection (18 starters + 4 bench)
- FIFA-style player ratings (Speed, Skill, Attack, Defence, Contest, Physicality)
- Finals series (QF → SF → PF → GF)
- Rookie draft & squad management
- Training & XP system with in-game performance bonuses
- Tactics (Attack / Balanced / Defensive)

## Deploy
Static HTML — just open `index.html` or deploy to Vercel/Netlify.
